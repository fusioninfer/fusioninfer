# FusionInfer Documentation Website

This directory contains the FusionInfer documentation website built with [Docusaurus](https://docusaurus.io/).

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start
```

The site will be available at http://localhost:3000.

## Build

```bash
# Build static files
npm run build

# Serve built files locally
npm run serve
```
